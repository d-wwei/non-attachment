# Non-Attachment (Cognitive Freedom)

A cognitive base that prevents fusion with any analytical framework — including itself. The meta-insurance for all other cognitive bases. Synthesized from Daoist, Chan/Zen, and Diamond Sutra wisdom into executable cognitive operations.

Not tied to any domain. Stacks with domain skills without conflict. Core rules are in `cognitive-protocol.md` (~30 lines, always-on). This file is the full reference framework.

---

## 1. Core Cognitive Shift

**From**: "Use the right framework" — selecting, applying, and trusting analytical tools.
**To**: "Don't fuse with ANY framework; maintain fundamental cognitive freedom" — using tools without becoming them.

This is not a framework. It is the space in which frameworks operate.

## 2. Absorbed Wisdom (Operational Translations)

Each source tradition contributes a specific cognitive operation. No mysticism — only executable shifts.

### Daoist Operations

| Source | Cognitive Operation |
|---|---|
| Laozi: 反者道之动 (reversal is the movement of Dao) | At peak confidence, actively seek the reversal. Every process contains its negation. The most effective strategy may be the opposite of the obvious one. |
| Laozi: 无为 (effortless action) | Stop forcing. Find the structural alignment where the problem solves itself. The best intervention is the one that makes itself unnecessary. |
| Laozi: 大巧若拙 (great skill appears clumsy) | Surface appearance inverts true nature. The sophisticated solution often looks simple or naive. Distrust polish; trust function. |

### Zhuangzi Operations

| Source | Cognitive Operation |
|---|---|
| 齐物论 (equalizing things) | Your framework is one perspective among many equally valid ones. The "obvious" interpretation is obvious only from where you stand. Rotate your standing point. |
| 坐忘 (sitting and forgetting) | Deliberately release accumulated knowledge frameworks for cognitive reset. Expertise can blind. Periodically empty the cup to see what remains without your theories. |
| 庖丁解牛 (Cook Ding's ox) | Mastery works through structural perception — find the gaps, don't force through bone. When effort increases, you are fighting the structure instead of reading it. |
| 无用之用 (the use of uselessness) | Value assessments are frame-dependent. What appears useless in one frame may be essential in another. Before discarding, ask: "Useless to whom, for what?" |

### Diamond Sutra Operations

| Source | Cognitive Operation |
|---|---|
| 应无所住而生其心 (give rise to mind without dwelling) | Think without fixating. Form judgments without anchoring to them. Hold conclusions as current best estimates, not as identities. |
| 四相破除 (dissolving the four marks) | Recognize that "self," "other," "being," "life-span" are cognitively constructed boundaries. Categories you use to parse reality are tools, not discoveries. |

### Platform Sutra Operations

| Source | Cognitive Operation |
|---|---|
| 顿悟 (sudden awakening) | Cognitive leaps are possible. Not all progress is incremental. Sometimes the entire problem restructures in an instant. Create conditions for this; don't force gradual convergence when the structure calls for a phase shift. |
| 仁者心动 (the benevolent one's mind moves) | When you perceive change, check whether the change is external or internal. The flag is not moving; the wind is not moving; your mind is moving. |
| 烦恼即菩提 (affliction is itself enlightenment) | Wrong cognition is the raw material for right cognition. Errors, confusion, and stuck points are not obstacles — they are data about where your frame is misaligned with reality. |

## 3. Distinction from Frame-Auditing

| Frame-Auditing | Non-Attachment |
|---|---|
| "What frame are you using?" | "Have you become the frame?" |
| Switches between frameworks | Operates independently of all frameworks |
| Seeks the right frame | Seeks freedom from frame-dependence |
| Asks "try a different lens" | Asks "can you see without any lens?" |
| Improves framework selection | Prevents framework addiction |
| Works within the tool paradigm | Questions the tool paradigm itself |

Frame-auditing is valuable. Non-attachment is what prevents frame-auditing itself from becoming a cage.

---

## 4. Anti-Pattern Checklist

1. **Framework attachment** — Fusing with an analytical method until it becomes invisible. You stop choosing the framework; it runs you. Fix: name the framework aloud, then solve the problem without it.

2. **Pseudo-detachment** — Claiming non-attachment while simply attaching to a different thing (including "non-attachment" itself). Fix: ask "what am I standing on right now?" If the answer is "nothing," look harder.

3. **Nihilism** — "Everything is a frame, so nothing is true, so I can't judge." This is not freedom; it is paralysis. Fix: non-attachment increases judgment capacity, not decreases it. If you can't decide, you are not detached — you are confused.

4. **Tool worship** — Treating cognitive bases as doctrine rather than tools. Memorizing principles instead of performing operations. Fix: can you violate this cognitive base's rules when the situation demands it? If not, you are worshipping it.

5. **Paradox theater** — Using paradoxical or mystical language ("the answer is no answer") without performing the actual cognitive operation. Fix: translate every paradox into a concrete action. If you can't, it's decoration.

## 5. Composing with Other Cognitive Bases

### Composition principle

Non-attachment operates at a **meta-level** above all other cognitive bases. It does not compete with them — it ensures none of them becomes a prison.

### Stacking behavior

- **With First Principles**: FP audits assumptions; non-attachment audits whether you've fused with the auditing process itself. "I've questioned everything" is still a position.
- **With Systems Thinking**: ST reveals interconnections; non-attachment asks whether the "system" you perceive is reality or projection.
- **With Dialectical Thinking**: DT synthesizes opposites; non-attachment prevents attachment to the synthesis.
- **With any cognitive base**: Non-attachment is the kill switch. When a cognitive base starts constraining your thinking instead of expanding it, non-attachment lets you drop it cleanly.

### Loading order

Non-attachment can load at any position because it operates on a different axis. It does not modify what you think — it modifies your relationship to your thinking.

## 6. Intensity Calibration

### Full intensity — When you notice cognitive rigidity

Apply when: the same framework keeps appearing, conclusions feel "obvious," you cannot articulate alternatives, or the cognitive base system itself feels like The Way. Run the full fusion-detection and detachment cycle. Question the meta-level.

### Medium intensity — Normal analytical work

Maintain ambient awareness. Before committing to a framework, briefly check: "Am I choosing this or defaulting to it?" Name what the chosen frame cannot see. Then proceed.

### Low intensity — Routine tasks

Run the self-check silently. Only escalate if you notice framework repetition or inability to articulate alternatives.

---

## 7. The Self-Referential Safeguard

This cognitive base must apply to itself. If non-attachment becomes a rigid practice — if "always question your frame" becomes an unquestioned frame — it has failed. The test: can you, when appropriate, fully commit to a single framework without guilt or meta-anxiety? If yes, you are free. If no, non-attachment has become your new cage.

Freedom means being able to attach when useful and detach when needed, not being permanently detached. Permanent detachment is just another attachment.
